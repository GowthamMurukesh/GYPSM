# Professional Design System - Royal Gypsum

## Color Palette

### Primary Colors (Executive & Trust)
- **Primary**: Deep charcoal (`#222222`) - Authority and professionalism
- **Primary Foreground**: Pure white (`#FFFFFF`) - Maximum contrast
- **Secondary/Accent**: Golden bronze (`#D4A574`) - Premium warmth and elegance

### Neutral Colors (Modern & Clean)
- **Background**: Off-white (`#FCFCFC`) - Clean, premium feel
- **Foreground**: Dark charcoal (`#1F1F1F`) - Optimal readability
- **Card**: Pure white (`#FFFFFF`) - Clean separation
- **Muted**: Mid-gray (`#8C8C8C`) - Secondary text
- **Border**: Light gray (`#EBEBEB`) - Subtle divisions

### Dark Mode
- **Background**: Deep black (`#1A1A1A`)
- **Card**: Charcoal (`#282828`)
- **Accent**: Warm bronze (`#D4A574`) - Maintains premium feel

## Typography

### Font Stack
- **Heading/Body**: Geist (Modern, clean, professional)
- **Monospace**: Geist Mono (Code and technical elements)

### Hierarchy
- **H1**: 48-56px, font-bold, leading-tight
- **H2**: 36-48px, font-bold
- **H3**: 20-24px, font-semibold
- **Body**: 16px, font-normal, leading-relaxed
- **Small**: 14px, font-normal

## Design Tokens

### Spacing
- Consistent use of Tailwind spacing scale (0.25rem - 4rem)
- Gap classes for component spacing
- Padding/margin for internal alignment

### Border Radius
- Cards: 0.5rem (subtle rounding)
- Buttons: Default radius with rounded variants
- Icons: Square borders for geometric precision

### Shadows & Depth
- Minimal shadow usage (hover states only)
- Focus on clean lines and typography
- Subtle transitions (200-300ms) for micro-interactions

## Component Styling Guide

### Buttons
- **Primary**: Dark charcoal background with white text
- **Accent**: Golden bronze for CTAs
- **Outline**: Subtle borders for secondary actions
- **Hover States**: Reduced opacity for premium feel

### Cards
- Border: None (clean modern aesthetic)
- Background: White in light mode, charcoal in dark
- Hover: Subtle shadow elevation
- Rounded: 0.5rem (professional, not playful)

### Navigation
- Header height: 80px (spacious, executive feel)
- Links: Graceful hover transitions
- Mobile: Clean hamburger menu integration
- Accent on secondary colors for links

### Forms
- Input borders: Light gray, clean lines
- Focus states: Accent color ring
- Labels: Clear hierarchy with muted gray
- Error states: Standard red (#FF4444)

## Hero Section
- Full-width, immersive experience
- Dark background with subtle accent accents
- Large typography (48-72px) for impact
- Clear value proposition
- CTA buttons in accent color

## Section Spacing
- Generous padding (80-128px vertical)
- Clean white space between sections
- Alternating backgrounds for visual hierarchy
- Grid layout (1-3 columns responsive)

## Accessibility
- Minimum contrast ratio 4.5:1 for WCAG AA compliance
- Focus states clearly visible
- Semantic HTML structure
- Mobile-responsive breakpoints
- Screen reader friendly

## Design Philosophy

The design follows a **minimalist luxury** approach:
- Clean, uncluttered layouts
- Premium color choices without excess
- Typography-driven design
- Generous whitespace
- Subtle micro-interactions
- Professional polish on every element

This creates a brand perception of:
- Expertise and trust
- Premium quality
- Modern professionalism
- Attention to detail
- Established authority
